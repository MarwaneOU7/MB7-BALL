    🎮 COMMANDES DU PROGRAMME
    ═══════════════════════════════════════════════════════════

    CLIC GAUCHE     → Ajouter une balle à la position du curseur
    R               → Réinitialiser (supprimer toutes les balles)
    D               → Afficher/masquer le debug (FPS + compteur)
    ESC             → Quitter le programme proprement


    🔨 COMMANDES DE COMPILATION
    ═══════════════════════════════════════════════════════════

    make                → Compiler le projet
    make run            → Compiler et lancer
    make debug          → Compiler en mode debug
    make clean          → Supprimer fichiers compilés
    make install-raylib → Installer Raylib (Linux/macOS)


    🏗️ ARCHITECTURE EN 30 SECONDES
    ═══════════════════════════════════════════════════════════

    main.c
      ↓
      └─→ Game_Init()        [Initialiser]
      └─→ Game_Run()         [Boucle 60 FPS]
          ├─ HandleInput()   [Clavier + souris]
          ├─ Update()        [Physique]
          │  ├─ Appliquer gravité
          │  ├─ Mettre à jour positions
          │  ├─ Gérer rebonds
          │  └─ Détecter collisions
          └─ Draw()          [Affichage]
      └─→ Game_Cleanup()     [Libérer mémoire]


    📊 PHYSIQUE EN 4 FORMULES
    ═══════════════════════════════════════════════════════════

    1️⃣ Gravité:       vy += 0.4           (accélération vers le bas)
    2️⃣ Rebond:        v = -v * 0.95       (inversion + perte énergie)
    3️⃣ Friction:      v *= 0.999          (ralentissement progressif)
    4️⃣ Collision:     Échange d'énergie   (conserver le momentum)
