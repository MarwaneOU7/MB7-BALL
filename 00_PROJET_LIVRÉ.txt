🚀 PROCÉDURE DE COMPILATION
═══════════════════════════════════════════════════════════════════════════════

ÉTAPE 1: Installer Raylib (5-10 minutes)
──────────────────────────────────────────
  Windows:
    - Télécharger MSYS2: https://www.msys2.org/
    - Installer dans C:\msys64\
    - Exécuter INSTALL_SIMPLE.md ou INSTALL_WINDOWS.bat
  
  Linux:
    - Exécuter: ./install_linux.sh (avec chmod +x avant)
    - OU: sudo apt-get install libraylib-dev
  
  macOS:
    - brew install raylib


ÉTAPE 2: Compiler (< 1 minute)
───────────────────────────────
  Windows:
    - Ouvrir "MSYS2 MinGW 64-bit"
    - cd /c/Users/Marouan/PROJECT\ JEUX/bouncing_balls
    - mingw32-make run
  
  Linux/macOS:
    - cd ~/PROJECT\ JEUX/bouncing_balls
    - make run


ÉTAPE 3: Lancer (instant)
──────────────────────────
  ✓ La fenêtre s'ouvre automatiquement
  ✓ 3 balles apparaissent
  ✓ Cliquer pour ajouter des balles
  ✓ Les balles rebondissent réaliste
